* Language server for Python

See the Emacs documentation at [[https://emacs-lsp.github.io/lsp-pyright/][lsp-pyright]].
#+begin_src emacs-lisp
  (use-package lsp-pyright
    :ensure t
    :hook (python-mode . (lambda ()
                           (require 'lsp-pyright)
                           (lsp))))
#+end_src
